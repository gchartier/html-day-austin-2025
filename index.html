What is this?

HTML is the most basic building block of the Web. It forms the structure and meaning of every website you've ever visited.

HTML Day is an annual celebration of HTML, where people around the world gather IRL to write and learn HTML.

The goal is to make the web more human, handmade, and accessible to everyone.

Who is it for?

Literally anyone. No coding skills or prior experience required.

Everyone can learn to code in HTML. If you can read and write (or type) then you can write HTML!

Don't worry if you don't know HTML, we'll all help each other out. üòÉ
